Why Learn SQL?

Data is the fuel that power the businesses today. Data is used to inform
companies about how their strategy, marketing, operations are going. Having the ability to manipulate data will give you great employable skill.

SQL stand for Structured Query Language. It is used to retrieved and manipulate data in a relational database. SQL allow user to perform CRUD operations on data. CRUD stands for Create, Read, Update, Delete.

Let's get started!

To select a row from a table.
select * from table_name; * Means select every column in the table; from indicates which table to retrieve the data from. and table_name specifies the name of the table.

If you want to retrieve data from a specific colum the syntax looks something like this:
Select colum_name from table_name; select first_name from employees;

To select multiple column from a table: select column_1, column_2 from table_name;
you must use comma to separate the names of the columns select first_name, last_name from employees;
